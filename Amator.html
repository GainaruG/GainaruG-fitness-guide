Program de Antrenament pentru Amatori(Barbati)

Ziua 1: Antrenament pentru piept și triceps

  1. Bancă plano: 4 seturi x 8-10 repetări
  2. Flotări: 3 seturi x maximul de repetări (sau flotări cu genunchii pe podea)
  3. Flotări la aparat: 3 seturi x 10-12 repetări
  4. Presa la bancă orizontală cu gantere: 3 seturi x 10-12 repetări
  5. Extensii triceps cu bară EZ: 3 seturi x 8-10 repetări
  6. Extensii triceps cu cablu cu prindere V: 3 seturi x 10-12 repetări

Ziua 2: Antrenament pentru spate și biceps

  1. 1. Tracțiuni la bară fixă sau cu priză inversă: 4 seturi x maximul de repetări
  2. 2. Pulldown la aparat cu priză largă: 3 seturi x 10-12 repetări
  3. 3. Rămășițe cu gantere: 3 seturi x 10-12 repetări
  4. 4. Bară pentru biceps cu gantere: 3 seturi x 8-10 repetări
  5. 5. Flexii la cablu cu prindere V pentru biceps: 3 seturi x 10-12 repetări

Ziua 3: Odihnă

Ziua 4: Antrenament pentru picioare

  1. Presa pentru picioare: 4 seturi x 8-10 repetări
  2. Extensii pentru picioare: 3 seturi x 10-12 repetări
  3. Genuflexiuni cu bara: 3 seturi x 8-10 repetări
  4. Genuflexiuni hack sau lungi: 3 seturi x 10-12 repetări
  5. Lunges cu gantere: 3 seturi x 10-12 repetări
  6. Flexii pentru gambe la aparat: 4 seturi x 12-15 repetări

Ziua 5: Antrenament pentru umeri

  1. Ridicări laterale cu gantere: 4 seturi x 10-12 repetări
  2. Press militar cu gantere sau la aparat: 3 seturi x 8-10 repetări
  3. Ridicări frontale cu gantere: 3 seturi x 10-12 repetări
  4. Flotări pentru umeri cu gantere: 3 seturi x 10-12 repetări
  5. Shrugs cu gantere sau la aparat: 4 seturi x 10-12 repetări

Ziua 6: Odihnă

Ziua 7: Antrenament pentru brațe

  1. Biceps: Bară EZ curl sau biceps cu gantere: 4 seturi x 8-10 repetări
  2. Triceps: Extensii triceps la cablu cu bară V: 4 seturi x 10-12 repetări
  3. Biceps: Flexii concentrat cu gantere: 3 seturi x 10-12 repetări
  4. Triceps: Presă triceps cu bară sau la aparat: 3 seturi x 8-10 repetări
  5. Biceps: Flexii la cablu cu prindere inversă: 3 seturi x 10-12 repetări
  6. Triceps: Extensii triceps cu gantere: 3 seturi x 10-12 repetări
